NeoMusic ("kami", "milik kami", atau "aplikasi") dibangun dengan mengutamakan privasi. Kami tidak menjalankan analitik, kami tidak melacak Anda, dan kami tidak mengoperasikan server apa pun yang menyimpan data Anda. Kebijakan ini menjelaskan sedikit kasus, yang sepenuhnya bersifat opsional, di mana aplikasi berkomunikasi dengan internet, dan apa yang terjadi pada informasi Anda dalam setiap kasus.

Kebijakan ini berlaku untuk semua distribusi NeoMusic, termasuk **Google Play** dan **Huawei AppGallery**.

---

## 1. Data yang Kami Kumpulkan
Sesuai dengan prinsip **Minimalisasi Data (Pasal 5)** dari **Peraturan Perlindungan Data Umum (GDPR)**, NeoMusic dirancang untuk berfungsi tanpa mengumpulkan data pribadi.

* **Tanpa Data Pribadi:** Kami tidak mengumpulkan nama, alamat email, nomor telepon, atau alamat IP.
* **Tanpa Pelacakan:** Kami tidak menggunakan cookie, pengenal iklan, atau segala bentuk pelacakan perilaku.
* **Tanpa Analitik / Pelaporan Kerusakan:** Kami tidak menggunakan SDK analitik atau pelaporan kerusakan pihak ketiga (seperti Firebase atau Sentry).
* **Tanpa Akun:** NeoMusic tidak memerlukan atau menawarkan pendaftaran atau login apa pun.

## 2. Pemrosesan Lokal
NeoMusic berjalan terutama di perangkat. Data berikut disimpan **hanya di perangkat Anda** dan tidak pernah dikirimkan kepada kami:

* **Indeks Pustaka Media:** Aplikasi memindai penyimpanan perangkat Anda untuk mengindeks dan memutar file musik Anda.
* **Preferensi & Personalisasi:** Tema, warna aksen, pengaturan ekualiser, dan opsi lainnya.
* **Daftar Putar, Favorit & Statistik Mendengarkan:** Daftar putar, lagu favorit, dan riwayat mendengarkan Anda disimpan dalam basis data lokal di perangkat Anda.

## 3. Fitur Daring Opsional
Beberapa fitur mengakses internet **hanya ketika Anda mengaktifkan atau memicunya**. Fitur tersebut nonaktif atau sesuai permintaan secara bawaan dan dapat dinonaktifkan kapan saja.

* **Lirik Daring (LRCLib):** Ketika lirik daring diaktifkan, NeoMusic mengirimkan **judul lagu, artis, album, dan durasi** dari lagu yang sedang diputar ke [LRCLib](https://lrclib.net) untuk mencari lirik yang cocok. Tidak ada pengenal pribadi, data akun, atau pengenal perangkat yang dikirim, dan file audio itu sendiri tidak pernah diunggah. Lirik yang diambil disimpan dalam cache secara lokal. Anda dapat menonaktifkan fitur ini di **Pengaturan**.

## 4. Pembelian & Premium
NeoMusic menawarkan **pembelian premium sekali bayar** opsional untuk membuka kustomisasi tambahan. Kami tidak memproses atau menyimpan informasi pembayaran apa pun sendiri.

* Pembelian sepenuhnya ditangani oleh penyedia penagihan platform: **Google Play Billing** di Google Play, atau **Pembelian Dalam Aplikasi Huawei (IAP)** di Huawei AppGallery.
* Penyedia ini mengumpulkan dan memproses detail pembayaran berdasarkan **kebijakan privasi mereka sendiri**. Kami hanya menerima konfirmasi bahwa pembelian atau pemulihan berhasil, yang disimpan di perangkat Anda sebagai penanda premium lokal.

## 5. Izin Android
Untuk memberikan pengalaman ini, aplikasi mungkin meminta izin sistem berikut:

* **READ_MEDIA_AUDIO / READ_EXTERNAL_STORAGE:** Diperlukan untuk menemukan dan memutar file musik Anda.
* **Akses tulis ke file media (berdasarkan permintaan):** Ketika Anda menggunakan editor tag bawaan (untuk mengedit metadata atau sampul album) atau menghapus sebuah lagu, Android menampilkan dialog konfirmasi sistem untuk file tertentu tersebut. NeoMusic memodifikasi file hanya dengan tindakan eksplisit Anda.
* **Internet:** Digunakan semata-mata untuk pencarian lirik daring opsional dan untuk menyelesaikan pembelian melalui penyedia penagihan platform.

## 6. Hak Anda
Anda berhak untuk mengakses, memperbaiki, atau menghapus data Anda. Karena NeoMusic **tidak menyimpan data Anda di server kami**, hak-hak ini terpenuhi berdasarkan desain:

* **Hak untuk Menghapus:** Menghapus instalasi aplikasi akan menghapus semua preferensi, daftar putar, statistik, dan lirik yang di-cache yang tersimpan secara lokal.
* **Hak untuk Menolak:** Anda dapat mencabut izin media kapan saja melalui Pengaturan Sistem Android Anda, dan menonaktifkan lirik daring di dalam aplikasi.

## 7. Layanan Pihak Ketiga
NeoMusic tidak menggunakan jaringan periklanan atau layanan analitik. Satu-satunya pihak ketiga yang terlibat adalah:

* **LRCLib** ([lrclib.net](https://lrclib.net)) — untuk pencarian lirik daring opsional.
* **Penyedia penagihan** — Google Play atau Huawei IAP — untuk memproses pembelian premium opsional.

Di luar fitur opsional ini, data Anda tidak pernah meninggalkan perangkat Anda.

## 8. Privasi Anak-anak
NeoMusic tidak ditujukan untuk anak-anak dan tidak dengan sengaja mengumpulkan informasi pribadi dari siapa pun, termasuk anak-anak.
