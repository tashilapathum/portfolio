#### Tech Stack
- Kotlin for frontend logic with XML layouts for UI
- Firebase Auth, Storage and Firestore for backend
- Multi-module project with MVVM architecture
- Main app for Google Play and a separate admin app for operation management

#### App features

- A vast collection of PDF files wit ha built-in PDF viewer
- Online Exams to participate and compete against other users
- Daily leaderboard and rewards for top ranks
- Resources in both Sinhala and English languages
